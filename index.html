Introducere

În matematica recreationala, o matrice pătrată de numere, de obicei numere întregi pozitive, se numește pătrat magic dacă sumele numerelor din fiecare rând, fiecare coloană și ambele diagonale principale sunt aceleași. Ordinea pătratului magic este numărul de numere întregi de-a lungul unei laturi n, iar suma constantă se numește constantă magică(notata cu "M"). Dacă tabloul include doar numerele întregi pozitive , se spune că pătratul magic este normal.

Studiul matematic al pătratelor magice se ocupă de obicei de construcția, clasificarea și enumerarea lor. Deși nu există metode complet generale de producere a tuturor pătratelor magice de toate ordinele, din punct de vedere istoric s-au descoperit cateva tehnici generale : metoda la Loubre(numita uzual metoda siameza) si metoda rombului pentru patratele de ordin impar, metoda LUX pentru patrate de ordinul 4n+2 sau metoda "doubly even"(dublu par) pentru patrate de ordinul 4n.

Patrate magice de-a lungul istoriei

În China antică, se cunoșteau pătratele magice încă din Mileniul al III-lea î.Hr., după cum atestă Lo Shu. După legendă, într-o bună zi se revărsă un râu; oamenii, înfricoșați, încercară să aducă o ofrandă zeilor râului Lo (unul dintre cele revărsate) pentru a-i calma furia. Totuși, de fiecare dată când făceau aceasta, apărea o broască țestoasă care încercuia ofrandele fără să le accepte, până când un băiat își dădu seama de marcajele speciale de pe carapacea ei și așa putură să ofere cantitatea cerută (15), și să mulțumească zeul, care readuse apele la nivelul lor. Au cunoscut și combinații de această clasă indienii, arabii, egiptenii și grecii. La pătrate asemănătoare, diferitele culturi au atribuit proprietăți astrologice și divinatorii variate, fiind de numeroase ori marcate în talismane. Așa cum reia Cornelius Agrippa în "Despre filozofia ocultă III" (1533), pătratul de ordinul trei(M=15) era consacrat zeului Saturn, cel de patru(M=34) lui Jupiter, cel de cinci(M=65) lui Marte, cel de șase(M=111) Soare, cel de șapte(M=175) lui Venus, cel de opt(M=260) lui Mercur și cel de nouă(M=369) Lunei; o atribuție similară se poate găsi în astrologia hindusă.

Introducerea pătratelor magice în occident se poate atribui lui Emanuel Moschopoulos, în jurul secolului al XVI-lea, autorul unui manuscris în care pentru prima oară au fost explicate câteva metode pentru a le construi. Mai târziu, studiul proprietăților acestor pătrate a atras atenția unor mari matematicieni, care au dedicat subiectului câteva opere chiar cu toată inutilitatea practică a pătratelor magice. Printre ei se pot cita Stifel, Fermat, Pascal, Leibnitz, Frénicle de Bessy, Bachet de Méziriac, La Hire, Saurin, Euler, ... se poate zice că nici un matematician nu a putut rezista farmecelor pătratului magic.

De asemenea, pătratele magice și-au făcut apariția și în artă înca din secolul al XVI-lea :
- Pătratul magic(de ordin 4) al lui Albrecht Dürer, sculptat în opera sa Melancolía(1514) este considerat primul din artele europene,
- Fațada Patimilor a bisericii Sagrada Família în Barcelona, concepută de sculptorul Josep Subirachs arată un pătrat magic de ordinea 4: Constanta magică a pătratului este 33, vârsta lui Iisus Cristos în timpul Patimilor. Structural, este forte asemănătoare pătratului magic din Melancolia, dar două numere din pătrat, (12 și 16) sunt reduse cu două unități (10 și 14) iar de aceea apar repetiții. Aceasta permite să se reducă constanta magică cu 1.

Construcția pătratului de ordin impar

Patrate magice de ordin impar - metoda I

Aceste pătrate pot fi generate cu metoda publicată în 1691 de Simon de la Loubere, numită câteodată metoda siameză, metodă cunoscută de astrologii orientali. Începând în căsuța centrală a primului rând cu primul număr, umplem diagonala ruptă cu următoarele, în sens NV (sau NE). Odată umplută prima diagonală, este coborâtă de o poziție și se umple a doua în același sens ca și prima, apoi repetând pașii anteriori până se termină pătratul.


Metoda siameza poate fi transpusă astfel în cod :

#include <iostream>
using namespace std;
int n,nr,a[101][101];
int main()
{   cin>>n;
    for(int i=0,j=n/2+1,d=1;d<=n;d++)
    {   a[++i][j]=++nr;
        for(int k=1;k<n;k++)
        {   i--;
            if(i==0) i=n;
            j++;
            if(j==n+1) j=1;
            a[i][j]=++nr;
        }
    }
    for(int i=1;i<=n;i++)
    {   for(int j=1;j<=n;j++) cout<<a[i][j]<<' ';
        cout<<'\n';
    }
    return 0;
}
  • Se pornește din celula din mijloc a laturii de sus, pe direcția NE
  • Cand se ajunge pe prima linie/coloana, următoarea valoare marcată va fi pe aceeași direcție, dar pe ultima linie/prima coloana
  • Cand pe direcția aleasă se gaseste o celula care este deja populată, se coboara o linie si se continuă algoritmul

Construcția pătratului de ordin 4*n

Model genericToate numerele sunt scrise în ordine de la stânga la dreapta pe fiecare rând pe rând, începând din colțul din stânga sus. Numerele sunt apoi reținute în același loc sau schimbate cu numerele lor diametral opuse într-un anumit model regulat. În pătratul magic de ordinul patru, numerele din cele patru pătrate centrale și câte un pătrat la fiecare colț sunt reținute în același loc, iar celelalte sunt schimbate cu numerele lor diametral opuse.

O construcție a unui pătrat magic de ordinul 4 Începând din stânga sus, mergeți de la stânga la dreapta prin fiecare rând al pătratului, numărând fiecare celulă de la 1 la 16 și umplând celulele de-a lungul diagonalelor cu numărul corespunzător. Odată ce ați ajuns la celula din dreapta jos, continuați mergând de la dreapta la stânga, începând din dreapta jos a tabelului prin fiecare rând și completați celulele non-diagonale numărând de la 1 la 16 cu numărul corespunzător. Așa cum se arată mai jos:

Codul caracteristic acestei metode arată astfel :

#include <iostream>
using namespace std;
int n,nr,a[101][101];
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
        for(int j=1;j<=n;j++)
    {
        nr++;
        if((i%4==1||i%4==0)&&(j%4==1||j%4==0))a[i][j]=nr;
        else if((i%4==2||i%4==3)&&(j%4==2||j%4==3))a[i][j]=nr;
        else a[i][j]=n*n-nr+1;
    }
    for(int i=1; i<=n; i++)
    {
        for(int j=1; j<=n; j++) cout<<a[i][j]<< ' ';
        cout<< '\n';
    }
    return 0;
}
  • Pătratul de ordin 4*n se împarte în pătrate de ordin 4,iar variabila nr reprezintă numărul de ordine al celulei curente
  • Fiecare celula marcată în ordine crescătoare va avea proprietățile :
    - i = 4*k+1 sau i = 4*k și j = 4*k+1 sau j = 4*k , unde k={0,1,2....n/4}(celulă din colț)

    sau

    - i = 4*k+2 sau i = 4*k+3 și j = 4*k+2 sau j = 4*k+3 , unde k={0,1,2....n/4}(celulă centrală)
  • Se marchează restul celulelor cu numărul complementar lui nr, adică n*n-nr+1
Construcția pătratului de ordin 4*n+2

Pentru a construi această clasă de pătrate magice se poate folosi metoda LUX. Se bazează în parte pe metoda lui la Loubere, care se folosește în construcția pătratelor magice de ordin impar.

Astfel, pentru construcția unui patrat de ordinul 10 se va folosi urmatorul algoritm :

Pasul întâi: Regrupăm căsuțele în grupuri de 2x2, și le etichetăm pe fiecare în parte cu forma următoare:
-Pătratele din primele k+1 rânduri, unde k este împărțirea completă a mărimii pătratului în patru, sunt etichetate cu litera L (3 rânduri în cazul acesta).
-Pătratele rândului următor se etichetează cu litera U.
-Pătratele rândurilor rămase se etichetează cu litera X.
Aceste litere ne vor arăta pe urmă cum să umplem fiecare pătrat de 2x2.

Algoritmul C++ corespunzător acestei metode este :

#include <iostream>

using namespace std;

int n;
int a[101][101];
void gen_impar(int n,int a[][101]){

    int nr=0;
    for(int i=0,j=n/2+1,d=1;d<=n;d++)
    {   a[++i][j]=++nr;
        for(int k=1;k<n;k++)
        {   i--;
            if(i==0) i=n;
            j++;
            if(j==n+1) j=1;
            a[i][j]=++nr;
        }
    }
}

int main()
{
    int b[101][101];
    cin>>n;
    gen_impar(n/2, b);
    /// generam matricea tip
    char tip[51][51];
    int k = (n-2)/4;
    for(int i=1;i<=k+1;++i)
        for(int j=1;j<=n/2;++j)
            tip[i][j] = 'L';
    for(int j=1;j<=n/2;++j)
        tip[k+2][j] = 'U';
    for(int i=k+3;i<=n/2;++i)
        for(int j=1;j<=n/2;++j)
            tip[i][j] = 'X';
    swap(tip[k+1][k+1], tip[k+2][k+1]);
    /// generam matricea rezultat
    for(int i=1, x=1;i<=n;i+=2, ++x)
        for(int j=1, y=1;j<=n;j+=2, ++y)
    {
        if(tip[x][y] == 'L')
        {
            a[i][j] = 4*b[x][y];
            a[i+1][j+1] = 4*b[x][y]-1;
            a[i+1][j] = 4*b[x][y]-2;
            a[i][j+1] = 4*b[x][y]-3;
        }
        else if(tip[x][y] == 'U')
        {
            a[i][j] = 4*b[x][y]-3;
            a[i+1][j+1] = 4*b[x][y]-1;
            a[i+1][j] = 4*b[x][y]-2;
            a[i][j+1] = 4*b[x][y];
        }
        else if(tip[x][y] == 'X')
        {
            a[i][j] = 4*b[x][y]-3;
            a[i+1][j+1] = 4*b[x][y]-2;
            a[i+1][j] = 4*b[x][y]-1;
            a[i][j+1] = 4*b[x][y];
        }
    }
    for(int i=1;i<=n;++i, cout<<'\n')
        for(int j=1;j<=n;++j)
            cout<<a[i][j]<<' ';
    return 0;
}
  • Matricea tip va reține tipul fiecărei diviziuni de 2x2 din matricea inițială(pasul 1 si 2)
  • Functia gen_impar va genera în matricea b numărul fiecarui subpătrat(pasul 3)
  • La sfarsit, vom popula matricea rezultat(matricea a) în funcție de fiecare tip de subpătrat(pasul 4)
